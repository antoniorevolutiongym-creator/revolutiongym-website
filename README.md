# Revolution Gym Bacoli Website

Modern premium homepage for Revolution Gym Bacoli, built with Next.js 15, React, TypeScript and Tailwind CSS.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run typecheck
```

## Project structure

- `src/app` — App Router pages, layout and global styles.
- `src/components` — Reusable UI components.
- `src/lib` — Shared content and utilities.
